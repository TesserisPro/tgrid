####data-g-enable-grouping

This property is used to disable sorting for this column.

**Value:** *true* or *false*.

**Default value:** *true*.

**Example**:
<!--Start the highlighter-->
<pre class="brush: html">
	<column data-g-member="Name" data-g-enable-grouping = "false"> </column>
</pre>
