####data-g-enable-filtering

This property is used to disable filtering for this column.

**Value:** *true* or *false*.

**Default value:** *true*.

**Example:**
<!--Start the highlighter-->
<pre class="brush: html">
	<column data-g-member="Name" data-g-enable-filtering = "false"> </column>
</pre>
